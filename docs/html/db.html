find_competencies_by_text_query(text_search_query: str) → List[Competency]¶

Find all competencies by text query. The competencies will be retrieved either by searching for an exact match in the competency description (case-sensitive in this case), or by using the preprocessing pipeline to preprocess the text and then search for all labels that contain the preprocessed search query and returning their associated competencies.

Parameters:

text_search_query (str) – sequence of words
