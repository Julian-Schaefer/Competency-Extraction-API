machine_learning.py¶

This file contains the method that has been used to train our Machine Learning model. Therefore, this file is a “standalone” module, which is not used by other parts of the system. Its sole purpose is for training the Machine Learning Model based on the annotated course descriptions that have been generated by our reference implementation of the Competency Extractor. More information regarding the training process can be found in the project report.

app.machine_learning.create_train_and_test_spacy_files(frac=0.8) → None¶

Creates spacy training and testing files for training a NER model. The data used to make the files comes from the courses_preprocessed.csv (location specified in “COURSES_FILE” environment variable) which contains the competencies that the Reference Implementation extracted from the course descriptions provided by the Weiterbildungsdatenbank Berlin Brandenburg. The files are saved in the in the sub directory “ML”.

Parameters:

frac (int) – The fraction of training data.
