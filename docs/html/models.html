class app.models.Course(id: int, description: str, extractor: str, competencies: List[Competency] = [])¶

Bases: object

Defines the data structure for storing and working with Courses. A Course only conists of a description, an id and the type of Competency Extractor that has been used to extract Competencies from the description. Optionally Courses can be related to multiple Competencies.
